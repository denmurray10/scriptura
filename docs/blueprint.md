# **App Name**: EchoForge

## Core Features:

- Character Creation: Allow users to design unique characters with customizable traits and backstories.
- Scenario Builder: Enable users to define interactive scenarios and story paths with choices and consequences.
- Dynamic Story Progression: Implement a system that dynamically adjusts the story based on user choices, AI-generated events, and character interactions.
- AI Visual Integration: Generate character portraits and scenario visuals, and enable LLM as tool which assists story and character developments using AI to match text to image. Use Deepseek AI instead of Gemini.
- AI-Driven Interactions: LLM analyzes user choices and determines their effects on character relationships, story progression, and event outcomes. Use Deepseek AI instead of Gemini.
- Story Export and Share: Allow users to export their created stories in various formats.

## Style Guidelines:

- Dark gray (#303134) for the main background to maintain a professional and focused environment.
- Teal (#26A69A) as the primary color for interactive elements to draw user attention.
- A lighter gray (#5F6368) for secondary elements to ensure visual hierarchy and reduce clutter.
- Roboto (sans-serif) for all text to ensure clarity and readability across the UI.
- Use simple, outlined icons in teal (#26A69A) for interactive elements.
- A clean, card-based layout to keep the content organized and easy to navigate.
- Subtle animations and transitions for feedback during user interactions to indicate progress.
